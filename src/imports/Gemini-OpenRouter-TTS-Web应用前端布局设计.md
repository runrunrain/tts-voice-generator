## 0. 版本变更说明

### 0.1 为什么推翻前一版

前一版（v1.0）采用 960px 居中文章式布局 + 三端响应式（desktop/tablet/mobile），存在三个根本性问题：

| 问题 | 分析 | 修正方向 |
|------|------|----------|
| 960px 居中在 1920px 桌面上大量留白 | TTS 工具不是博客文章。48% 的屏幕被空白浪费，核心操作区被压缩到窄条，信息密度极低。在 2560px 显示器上更严重 | 全窗口桌面工作台布局，充分利用每一像素 |
| 移动端/平板端设计毫无必要 | 本工具监听 127.0.0.1，用户在本地桌面上使用。不存在"在手机上打开本地服务"的真实场景。960px 居中本是为移动端下折列准备的妥协 | 删除全部移动端/平板端/底部导航设计 |
| 单列垂直滚动不符合专业工具心智 | 用户期望的是"工作站"而非"网页"。录音软件、视频编辑器、IDE 都是多面板并行布局，不是一列到底的文档流 | 多面板桌面工作台：导航 Rail + 中心工作区 + 右侧 Inspector/Output |

### 0.2 新版核心改动

| 维度 | v1.0 | v2.0 |
|------|------|------|
| 布局范式 | 960px 居中，单列滚动 | 全窗口桌面工作台，多面板并行 |
| 分辨率基准 | 1024px 起步，移动端 768px 以下 | 主设计 1920x1080，兼容 1366x768 - 2560x1440 |
| 导航方式 | 顶栏水平导航 + 移动端底部导航 | 左侧 Navigation Rail 72px |
| 内容展示 | 居中单列堆叠 | 中心工作区 + 右侧面板，按页面特性分配 |
| 结果/输出位置 | 按钮下方，垂直堆叠 | 右侧 Output 面板，输入与输出并排 |
| 状态信息 | 无固定展示位置 | 底部状态栏 28px，常驻显示连接/服务/生成状态 |
| 移动端 | 三断点响应式 | 不设计 |

## 1. 设计方向

### 1.1 视觉风格：深色录音控制台（Dark Console Workstation）

本工具的视觉身份是"专业本地创作软件"，不是"AI SaaS 网站"。用户打开时应感觉启动了一款桌面应用，而非访问了一个网页。

| 感受 | 传达方式 |
|------|----------|
| 专业工具感 | 全窗口无留白、固定面板边界、信息密度精确控制、深色基调 |
| 温暖创造力 | 琥珀暖金作为唯一强调色，VU 表灯光感的点缀，深色中的呼吸 |
| 专注沉浸 | 左侧 Rail 极简图标、中心区域无干扰、右侧面板可折叠、状态栏不抢注意力 |

### 1.2 风格参照

参照对象不是网站，而是桌面软件：

| 参照 | 借鉴点 |
|------|--------|
| Ableton Live | 深色基调、面板分区、信息密度、左 Rail 导航 |
| DaVinci Resolve | 多面板并行、底部状态/时间线、右侧 Inspector |
| Figma | 左侧图层面板 + 中心画布 + 右侧属性面板的三栏结构 |
| Logic Pro | 深色控制台、琥珀色电平表、工具栏图标风格 |

### 1.3 不做什么

- 不使用紫色渐变、霓虹色、全息彩虹等"AI 未来感"视觉
- 不使用 Inter、Roboto、Arial、system-ui 作为主要字体
- 不做纯白底 + 蓝色按钮的企业管理后台风格
- 不做拟物化（木纹、旋钮、表盘）等过度装饰
- 不使用 emoji 作为功能图标
- 不做居中文章式布局、不做卡片瀑布流
- 不设计移动端、平板端、响应式底部导航

## 2. 全局布局架构

### 2.1 应用壳（Application Shell）

全窗口固定布局，不随内容滚动。整体由 CSS Grid 定义五大区域。

```
1920 x 1080 全窗口视图
+--+--------------------------------------------+------------------+
|  |  TopBar                          48px       |                  |
|  +--------------------------------------------+------------------+
|N |                                            |                  |
|a |                                            |   Right Panel    |
|v |           Center Workspace                 |   (Output /      |
|  |           (可独立滚动)                       |    Inspector)    |
|R |                                            |                  |
|a |                                            |                  |
|i |                                            |                  |
|l +--------------------------------------------+------------------+
|  |  Bottom Status Bar              28px        |                  |
+--+--------------------------------------------+------------------+
 72px          flex-1                              384px (可折叠)
```

### 2.2 区域坐标与尺寸

**1920x1080 基准尺寸表**

| 区域 | 起始坐标 | 尺寸 | 布局方式 | 说明 |
|------|---------|------|---------|------|
| Nav Rail | (0, 0) | 72px x 1080px | Flex column | 固定左侧，不滚动 |
| TopBar | (72, 0) | 1848px x 48px | Flex row | 面包屑 + 标题 + 状态灯 + 快捷操作 |
| Center Workspace | (72, 48) | 1464px x 1004px | 按页面定制 | 内容区，可独立滚动 |
| Right Panel | (1536, 48) | 384px x 1004px | Flex column | 可折叠，折叠后 Center 占满 |
| Bottom Status Bar | (72, 1052) | 1848px x 28px | Flex row | 连接状态 + 服务信息 + 生成计数 |

**1366x768 兼容尺寸表**

| 区域 | 尺寸 | 变化 |
|------|------|------|
| Nav Rail | 64px x 768px | 缩窄 8px |
| TopBar | 1302px x 48px | 自适应缩窄 |
| Center Workspace | ~982px x 692px | Right Panel 收窄为 320px |
| Right Panel | 320px x 692px | 缩窄 64px |
| Bottom Status Bar | 1302px x 28px | Nav Rail 右侧全宽 |

**2560x1440 兼容尺寸表**

| 区域 | 尺寸 | 变化 |
|------|------|------|
| Nav Rail | 72px x 1440px | 不变 |
| Center Workspace | ~2056px x 1364px | 自动扩展 |
| Right Panel | 432px x 1364px | 扩展为 432px，内容更宽松 |

### 2.3 Navigation Rail

Nav Rail 是全局左侧垂直导航，高 100%，宽 72px（1366x768 时 64px）。

```
+--------+
| [Logo] |  32x32 logo，居中，顶部 12px margin
|        |
|--------|  1px 分割线，margin 8px
|        |
| [wave] |  生成页图标
|  生成   |  文字标签，10px
|        |
| [clap] |  Director 图标
| 导演    |  当前页高亮：accent 色背景块
|        |
| [mic]  |  音色页图标
| 音色    |  非活跃态：中性色
|        |
| [clock]|  历史页图标
| 历史    |
|        |
| [gear] |  设置页图标
| 设置    |
|        |
|        |  flex-1 空白
|        |
| [bell] |  Agent 待确认通知（条件显示）
| 通知    |  有未读时显示红点
+--------+
```

**Rail 设计规则**

| 规则 | 值 |
|------|-----|
| 每项高度 | 56px（图标 20px + 文字 10px + 间距） |
| 活跃态背景 | `rgba(201,148,74,0.15)` 圆角矩形 |
| 活跃态图标/文字色 | `--color-accent` |
| 非活跃态 hover | `--color-bg-hover` 圆角矩形 |
| 图标尺寸 | 20px |
| 文字标签 | 10px, `--weight-medium` |
| 图标与文字间距 | 4px |
| 整体水平内边距 | 8px（图标区域居中） |

### 2.4 TopBar

TopBar 高 48px，位于 Nav Rail 右侧。

```
+--[breadcrumb]--+--[page title]--+---------------------------+--[status]--+--[action]--+
| 生成 > 单次    | 语音生成        | (弹性空白)                  | [API: OK]  | [快捷生成] |
+----------------+-----------------+----------------------------+------------+------------+
 左侧                              中间                         右侧
```

| 区域 | 内容 | 对齐 |
|------|------|------|
| 左侧 | 面包屑导航（二级以内） | flex-start |
| 中间偏左 | 当前页面标题 | h2, `--text-lg` |
| 右侧 | 服务状态灯 + 快捷操作按钮 | flex-end |

**TopBar 设计规则**

| 规则 | 值 |
|------|-----|
| 高度 | 48px |
| 内边距 | 0 20px |
| 底部边框 | 1px solid `--color-border-subtle` |
| 背景色 | `--color-bg-base`（与 Nav Rail 一致） |
| 状态灯 | 8px 圆点 + 文字标签，绿=`--color-success`，红=`--color-error`，灰=未配置 |
| 快捷操作 | ghost 按钮，不干扰主区域 |

### 2.5 Right Panel

Right Panel 宽 384px（1366x768 时 320px），可通过拖拽 Splitter 调整宽度。

**面板最小宽度**：280px。低于此值自动折叠。

**折叠/展开控制**：面板右上角折叠按钮，或拖拽 Splitter 至最小宽度以下。

**面板内容按页面变化**：

| 页面 | Right Panel 用途 | 折叠时行为 |
|------|-----------------|-----------|
| /generate | 生成输出区：加载态/播放器/下载/生成信息 | 输出显示在 Center 底部区域 |
| /generate/director | 预览区：组装提示词预览 + 输出播放 | 预览弹窗替代 |
| /voices | 音色 Inspector：选中音色详情/探针控制 | 详情内联展示 |
| /history | 记录预览：选中记录的快速播放/下载 | 跳转详情页 |
| /history/:jobId | (隐藏) 详情页占满 Center | N/A |
| /settings | (隐藏) 设置页占满 Center | N/A |

### 2.6 Bottom Status Bar

Bottom Status Bar 高 28px，固定于窗口底部，横跨全宽。

```
+--[connection]--+--[service]--+----------(flex-1)-----------+--[stats]--+--[env]--+
| 127.0.0.1:3000 | 服务运行中    |                             | 今日生成: 3| v0.1.0  |
+----------------+-------------+-----------------------------+-----------+---------+
```

| 区域 | 内容 |
|------|------|
| 左侧 | 服务地址（可点击复制） + 连接状态 |
| 右侧 | 今日生成次数 + 应用版本号 |

**Status Bar 设计规则**

| 规则 | 值 |
|------|-----|
| 高度 | 28px |
| 字号 | `--text-xs` (11px) |
| 字色 | `--color-text-tertiary` |
| 背景 | `--color-bg-sunken`（比主背景更深一级） |
| 顶部边框 | 1px solid `--color-border-subtle` |
| 内边距 | 0 16px |

### 2.7 层级体系

| 层级 | z-index | 用途 |
|------|---------|------|
| 基础内容层 | 0 | Nav Rail、Center Workspace、Right Panel、Bottom Bar |
| TopBar | 10 | 固定顶部栏 |
| Splitter 拖拽层 | 50 | 面板分割线拖拽时的覆盖层 |
| Popover / Dropdown | 200 | 下拉选择、标签面板弹出 |
| 模态遮罩 | 300 | Agent 确认弹窗背景 |
| 模态内容 | 310 | 弹窗本体 |
| Toast 通知 | 400 | 全局通知浮层 |

### 2.8 Splitter（面板分割线）

Right Panel 左侧的垂直分割线，可拖拽调整面板宽度。

| 属性 | 值 |
|------|-----|
| 宽度 | 4px（可视区域 1px，热区 4px） |
| 默认色 | transparent |
| hover 色 | `--color-accent` 半透明 |
| 拖拽中色 | `--color-accent` 实色 |
| cursor | `col-resize` |
| 面板最小宽度 | 280px |
| 面板最大宽度 | 560px |
| 双击 | 恢复默认宽度（384px） |

| `--duration-fast` | `160ms` | 面板展开/收起、下拉出现 |
| `--duration-normal` | `240ms` | 页面过渡、弹窗出现 |
| `--duration-slow` | `400ms` | Toast 消失、面板 resize 过渡 |
| `--easing-default` | `cubic-bezier(0.4, 0, 0.2, 1)` | 通用缓动 |
| `--easing-decelerate` | `cubic-bezier(0, 0, 0.2, 1)` | 入场动画 |
| `--easing-accelerate` | `cubic-bezier(0.4, 0, 1, 1)` | 退场动画 |

> 动效时长比 v1.0 整体缩短 20%。桌面工具的操作节奏比网页更快。

**动效原则**

- 面板 resize 使用 `--duration-slow` + CSS transition on width，拖拽时切为即时响应
- 骨架屏脉冲使用 `1.5s ease-in-out infinite`
- 成功态播放器出现使用 `--duration-normal` + `--easing-decelerate`，从 `opacity:0; transform:translateY(6px)` 到 `opacity:1; transform:translateY(0)`
- Toast 从右上角滑入，停留 2.5 秒后淡出
- 所有动效支持 `prefers-reduced-motion`：开启时所有过渡切为 `0ms`

## 5. 可访问性（Accessibility）

### 5.1 焦点顺序

全局焦点流：Nav Rail -> TopBar -> Center Workspace -> Right Panel -> Bottom Status Bar。

| 页面 | 关键焦点路径 |
|------|-------------|
| /generate | Nav Rail 项 -> TopBar -> 文本区 -> 音色下拉 -> 格式选择 -> 生成按钮 -> Right Panel 播放器/下载 |
| /generate/director | Nav Rail -> TopBar -> 编辑列分区(从上到下) -> 配置列 -> 底部操作栏 |
| /voices | Nav Rail -> TopBar -> 工具栏 -> Tab -> 卡片(逐个) -> Right Panel 操作 |
| /history | Nav Rail -> TopBar -> 过滤栏 -> 记录行(逐个) -> Right Panel |
| /settings | Nav Rail -> TopBar -> 表单项(从上到下，先左后右) -> 保存按钮 |

### 5.2 键盘操作

| 操作 | 按键 | 作用域 |
|------|------|--------|
| 激活按钮/链接 | `Enter` / `Space` | 全局 |
| 关闭弹窗/下拉 | `Escape` | 弹窗、下拉、Toast |
| 下拉选择 | 上/下方向键 | 音色下拉、格式选择 |
| Tab 遍历 | `Tab` / `Shift+Tab` | 全局 |
| Splitter 拖拽 | 左/右方向键 | 面板分割线（每次移动 10px） |
| 折叠/展开 Right Panel | `Ctrl+B` | 全局快捷键 |

### 5.3 弹窗焦点陷阱

```
弹窗打开:
  1. 保存当前焦点元素引用 (triggerRef)
  2. focus 移入弹窗第一个可聚焦元素
  3. Tab 在弹窗内循环

弹窗关闭:
  1. focus 返回 triggerRef
  2. Escape 等同于"取消/拒绝"
```

实现：`aria-modal="true"` + `role="dialog"` + `inert` 标记背景。

### 5.4 状态不只用颜色

| 状态 | 视觉 | 辅助技术 |
|------|------|----------|
| 验证状态 | 圆点 + 文字标签 | `aria-label` 含状态文字 |
| 生成结果 | 图标 + 文字 + 背景 | `role="status"` + `aria-live="polite"` |
| 错误提示 | 红色边框 + 感叹号 + 文字 | `role="alert"` + `aria-live="assertive"` |
| 成功提示 | 绿色背景 + 勾号 + 文字 | `role="status"` |

```typescript
// Splitter 拖拽核心逻辑：更新应用壳 CSS 变量，而不是直接写 RightPanel width
const MIN_PANEL_WIDTH = 280;
const MAX_PANEL_WIDTH = 560;
const DEFAULT_PANEL_WIDTH = 384;

function initSplitter(splitterEl: HTMLElement, shellEl: HTMLElement) {
  let startX: number;
  let startWidth: number;

  const getPanelWidth = () => {
    const raw = getComputedStyle(shellEl).getPropertyValue('--right-panel-width').trim();
    return Number.parseInt(raw, 10) || DEFAULT_PANEL_WIDTH;
  };

  const setPanelWidth = (width: number) => {
    shellEl.style.setProperty('--right-panel-width', `${width}px`);
  };

  const onMouseDown = (e: MouseEvent) => {
    startX = e.clientX;
    startWidth = getPanelWidth();
    splitterEl.classList.add('splitter--dragging');
    document.addEventListener('mousemove', onMouseMove);
    document.addEventListener('mouseup', onMouseUp);
    // 阻止选中文本
    document.body.style.userSelect = 'none';
  };

  const onMouseMove = (e: MouseEvent) => {
    const delta = startX - e.clientX; // 向左拖拽 = 面板变宽
    const newWidth = Math.min(MAX_PANEL_WIDTH, Math.max(MIN_PANEL_WIDTH, startWidth + delta));
    setPanelWidth(newWidth);
  };

  const onMouseUp = () => {
    splitterEl.classList.remove('splitter--dragging');
    document.removeEventListener('mousemove', onMouseMove);
    document.removeEventListener('mouseup', onMouseUp);
    document.body.style.userSelect = '';
  };

  // 双击恢复默认宽度
  splitterEl.addEventListener('dblclick', () => {
    setPanelWidth(window.innerWidth >= 2400 ? 432 : DEFAULT_PANEL_WIDTH);
  });

  splitterEl.addEventListener('mousedown', onMouseDown);
}
```
